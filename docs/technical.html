Come Funziona la Sicurezza

La tua privacy è la nostra priorità assoluta

Crittografia Locale

Tutti i dati sono criptati localmente sul dispositivo utilizzando algoritmi di crittografia moderni e sicuri. Nessun dato viene mai inviato a server esterni.

Autenticazione Biometrica

Sblocco dell'app tramite impronta digitale o riconoscimento facciale, se supportato dal dispositivo. Combinabile con PIN personalizzabile.

PIN Personalizzabile

Configura un PIN di sicurezza personalizzato per proteggere l'accesso all'app. Il PIN è obbligatorio e può essere modificato in qualsiasi momento.

100% Offline

L'app funziona completamente senza connessione internet. Nessuna comunicazione con server esterni, nessun tracking, nessuna condivisione di dati.

Privacy Schermo

Opzione per oscurare lo schermo quando l'app è in background, proteggendo i dati da occhi indiscreti. Attivabile dalle impostazioni.

Database Sicuro

Il database locale è protetto da crittografia e accessibile solo tramite autenticazione. I backup sono esportati in formato JSON criptato.

Dettagli Tecnici

Crittografia

Utilizziamo librerie di crittografia standard del settore per proteggere i dati a riposo. La chiave di crittografia è derivata dal PIN/biometria dell'utente.

Archiviazione

Tutti i dati sono memorizzati nella memoria interna del dispositivo Android, in aree protette del filesystem. Nessun dato viene memorizzato su SD card esterne.

Permessi

L'app richiede solo i permessi strettamente necessari: camera (per QR code e foto), storage (per salvataggio locale). Nessun permesso di rete o posizione.

Requisiti Minimi

Cosa serve per utilizzare CatechHub

Sistema Operativo

  • Android 10.0 (API Level 29) o superiore
  • Testato su Android 10, 11, 12, 13, 14
  • Compatibile con la maggior parte dei dispositivi moderni

Hardware

  • RAM: Minimo 2GB (raccomandati 3GB+)
  • Spazio di archiviazione: ~100MB per l'app
  • Spazio aggiuntivo per dati e allegati (variabile)

Periferiche

  • Camera (obbligatoria per QR code e foto)
  • Biometria (opzionale ma raccomandata)
  • Touchscreen

Archiviazione

  • Memoria interna richiesta
  • SD card non supportata per dati sensibili
  • Backup esportabili in formato JSON

Note di Compatibilità

CatechHub è stato testato su una vasta gamma di dispositivi Android. Se riscontri problemi di compatibilità, ti invitiamo a segnalarli tramite GitHub Issues per permetterci di migliorare l'app.
