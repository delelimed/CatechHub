Come installare

  1. Scarica il file .apk dalla sezione Assets dell'ultima release su GitHub
  2. Apri il file scaricato — Android potrebbe chiederti di abilitare l'installazione "da fonti sconosciute" o "da questa origine"
  3. Vai in Impostazioni ? Sicurezza (o Impostazioni ? App ? Accesso speciale ? Installa app sconosciute) e concedi il permesso al file manager o al browser che hai usato
  4. Torna al file .apk e toccalo per installare
  5. Al primo avvio, CatechHub ti guiderà con un setup rapido: nome, cognome, gruppo

Come verificare che l'APK sia sicuro

  • Codice aperto e verificabile — L'intero codice sorgente è pubblico su GitHub. Chiunque può esaminarlo, compilarlo e confrontare gli hash.
  • Build riproducibile — Le release sono generate dai workflow ufficiali di GitHub Actions. Puoi vedere il log di build per ogni release.
  • Nessun dato esce dal telefono — L'app non ha server, non raccoglie statistiche, non invia dati a terze parti. Tutto resta cifrato sul dispositivo.
  • Scarica solo da GitHub — L'unica fonte ufficiale è la pagina Releases di GitHub. Non scaricare APK da siti terzi.
  • Protezione runtime — L'app include freeRASP che rileva manomissioni, root e tentativi di hacking.

Domande frequenti

Perché CatechHub non è sul Play Store?

Pubblicare un'app sul Play Store ha un costo. La distribuzione tramite APK è più semplice, diretta e mi permette di offrire l'app gratuitamente senza costi nascosti.

I dati sono sicuri?

Assolutamente sì. CatechHub è stato progettato con la privacy al centro. I dati non vengono mai caricati su cloud o server esterni: restano solo sul tuo telefono. Il database è crittografato con AES-256-GCM, l'accesso è protetto da impronta digitale o PIN, e la sincronizzazione con altri catechisti avviene con crittografia end-to-end. In più, l'app blocca screenshot non autorizzati e si blocca automaticamente dopo 2 minuti di inattività.

Posso usare l'app senza internet?

Sì, CatechHub funziona completamente offline. Tutti i dati sono memorizzati sul telefono e non hai bisogno di connessione per nessuna operazione: aggiungere ragazzi, fare l'appello, gestire documenti, tutto funziona sempre. L'unica cosa che richiede una connessione è il download iniziale dell'APK e la sincronizzazione volontaria con altri catechisti (via Bluetooth, che non ha bisogno di internet).

Come faccio a sincronizzare con altri catechisti?

La sincronizzazione avviene peer-to-peer via Bluetooth. Un catechista genera un QR code con la chiave di sincronizzazione, l'altro lo inquadra con la propria app, e i dati vengono scambiati in modo sicuro e crittografato. Nessun server centrale, nessun account. Decidi tu con chi condividere e quali dati.

Posso recuperare i dati se cambio telefono?

Certo! CatechHub ha una funzione di backup crittografato che crea un file .catechhub protetto da password. Puoi copiare questo file sul nuovo telefono e ripristinare tutto: ragazzi, gruppi, presenze, documenti. Puoi anche sincronizzare i dati con un altro catechista che ha già il tuo gruppo, se preferisci.

Come posso segnalare un bug o un problema?

Dall'app: vai in Impostazioni ? Feedback e suggerimenti e compila il modulo di segnalazione (include automaticamente versione app e informazioni sul dispositivo). In alternativa, apri una issue su GitHub.

L'app è davvero gratis?

Sì, CatechHub è e resterà completamente gratuita. Non ci sono abbonamenti, acquisti in-app o versioni "pro". Il codice è open source (licenza MIT), chiunque può verificarlo, contribuire o creare una propria versione. La sviluppo nel mio tempo libero, per passione e per servizio alla comunità parrocchiale.

Serve una registrazione o un account?

No, nessuna registrazione. Scarichi l'APK, installi, apri e inizi subito. Niente email, niente password, niente account. L'app è pronta all'uso dal primo avvio, con un semplice setup guidato.
