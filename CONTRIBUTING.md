# Contribuire a CatechHub

Grazie per l'interesse nel contribuire a CatechHub! Questo progetto è open source e aperto ai contributi di tutti. Se modifichi il progetto, ti chiediamo solo di mantenere il riconoscimento originale e di rispettare la licenza MIT.

## Come contribuire

1. Apri un issue su GitHub per discutere la modifica prima di iniziare.
   - Problemi di sicurezza
   - Bug
   - Nuove funzionalità
   - Miglioramenti alla documentazione

2. Fai un fork del repository e crea un branch specifico per la tua modifica:
   ```bash
   git checkout -b feature/nome-funzionalita
   ```

3. Implementa le modifiche.
   - Segui lo stile di codice esistente.
   - Mantieni la struttura del progetto Flutter.
   - Aggiungi test quando possibile.

4. Invia una pull request al branch `main` del repository principale.
   - Descrivi chiaramente il problema risolto.
   - Elenca i cambiamenti principali.
   - Includi eventuali istruzioni per testare la modifica.

## Linee guida per le pull request

- Assicurati che la tua PR sia piccola e focalizzata su un singolo cambiamento.
- Fornisci screenshot o esempi se modifichi l'interfaccia utente.
- Se aggiungi una nuova funzionalità, verifica che non rompa il comportamento esistente.

## Stile del codice

- Mantieni la chiarezza e la leggibilità.
- Usa nomi di variabili significativi.
- Per Dart/Flutter, segui le convenzioni ufficiali del linguaggio e del framework.

## Test e verifica

- Se il cambiamento riguarda logica o sicurezza, aggiungi test automatici quando possibile.
- Esegui i test con:
  ```bash
  flutter test
  ```

## Licenza e riconoscimenti

Questo progetto è rilasciato sotto licenza MIT. Puoi usare, modificare e distribuire il codice liberamente, ma devi mantenere il copyright originale e la notifica della licenza.

Se contribuisci al progetto, il tuo nome o il tuo account GitHub possono essere riconosciuti nella history del repository e nel registro delle pull request.

## Contatto

Per contributi, domande o segnalazioni, usa le issue su GitHub o scrivi a:
- `CatechHub.app@proton.me`

Grazie per contribuire a rendere CatechHub più sicuro e utile per tutti!
