Che cos'è

Il Backup Crittografato ti permette di salvare l'intero database di CatechHub in un file .catechhub protetto da password e autenticazione biometrica. Puoi conservarlo al sicuro, spostarlo su un altro dispositivo o ripristinarlo in caso di necessità.

Come fare un backup

  1. Dal menu impostazioni, scegli "Crea backup"
  2. Confermi con la tua impronta digitale o riconoscimento facciale
  3. Inserisci una password (almeno 8 caratteri)
  4. Scegli quali moduli includere (di default sono tutti selezionati)
  5. L'app crea il file e ti mostra il progresso dell'operazione
  6. Puoi salvarlo sul dispositivo, condividerlo o inviarlo via email

Come ripristinare un backup

  1. Scegli "Ripristina backup" dal menu impostazioni
  2. Confermi con biometria e inserisci la password del backup
  3. L'app verifica che il file non sia stato manomesso
  4. I dati vengono importati senza sovrascrivere quelli più recenti

Niente perdita di dati

Quando ripristini un backup, CatechHub non si limita a sovrascrivere tutto. Confronta ogni singolo record: se nel backup c'è una versione più recente del dato, quella viene presa; altrimenti resta la tua versione locale. I record nuovi vengono aggiunti, quelli esistenti non vengono mai persi.

Sicurezza

  • Crittografia AES-256-GCM: il file di backup è cifrato con lo standard militare
  • Autenticazione biometrica: serve l'impronta o il volto per avviare export e import
  • Password obbligatoria: senza la password giusta, il backup è illeggibile
  • Verifica integrità: prima di importare, l'app controlla che il file non sia corrotto o manomesso
  • Se il file è danneggiato, l'import viene annullato automaticamente
